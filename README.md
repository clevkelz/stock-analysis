# Using Visual Basic (VBA) to Evaluate Green Companies Stock Prices for 2017 and 2018
## Overview of the Project
The return on stock prices from 2017 and 2018 for the following 12 green energy (also referred to as "Green Companies") was reviewed to show the differences in daily volume and return among these companies.  This analysis was also performed to show how VBA can be used to analyze and make conclusions from a large dataset, in this case, daily stock prices for 12 Green Companies over the span of two years.
